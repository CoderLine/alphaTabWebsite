## Description
This method initiates the rendering of the specified tracks of the given score. 

## Signatures

<TypeTable>
    <TypeRow type="js" name="function resizeRender(score, trackIndexes)" />
    <TypeRow type="net" name="void RenderScore(Score score, IList<double> trackIndexes)" />
    <TypeRow type="android" name="fun renderScore(score: alphaTab.model.Score?, trackIndexes: alphaTab.collections.DoubleList?): Unit" />
</TypeTable>

### Parameters

<ParameterTable>
    <ParameterRow platform="all" name="score" type="AlphaTab.Model.Score">
        The score of which the tracks should be taken for rendering.
    </ParameterRow>
    <ParameterRow platform="all" name="trackIndexes" type="int[]">
        The indexes to the tracks which should be layed out and rendered. 
    </ParameterRow>
</ParameterTable>


### Returns
Nothing. The render result is computed asynchronously and can be obtained via events like `partialRenderFinished`.
