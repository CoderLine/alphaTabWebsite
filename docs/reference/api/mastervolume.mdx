## Description
Gets or sets the master volume of the overall audio being played. The volume is annotated in percentage where 1.0 would be the normal volume and 0.5 only 50%.

## Types

<TypeTable>
    <TypeRow type="all" name="double" />
</TypeTable>

## Default Value

`1.0`
