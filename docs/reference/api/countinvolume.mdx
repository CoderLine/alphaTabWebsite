## Description
Gets or sets the volume of the metronome during the count-in of the song. By default the count-in is disabled but can be enabled by setting the volume different.

## Types

<TypeTable>
    <TypeRow type="all" name="double" />
</TypeTable>

## Default Value

`0.0`
