## Description
This event is fired when the rendering of the whole music sheet is starting. All
preparations are completed and the layout and render sequence is about to start. 

<EventDescription metadata={metadata} />
    
## Types

<TypeTable>
    <TypeRow type="js" name="function()" />    
    <TypeRow type="net" name="Action" />    
</TypeTable>

## Parameters

<ParameterTable>
    <ParameterRow platform="all" name="resize" type="bool">
        Whether the rendering is triggered from a resize.  
    </ParameterRow>
</ParameterTable>
