Clears the highlight markers marking the currently selected playback range. Unlike actually setting <Signature style="inline" js={[["identifier","playbackRange","/docs/reference/types/alphatabapibase/#playbackrange"]]}>playbackRange</Signature>  this method only clears the selection markers without actually
changing the playback range. This method can be used when building custom selection systems (e.g. having draggable handles).
