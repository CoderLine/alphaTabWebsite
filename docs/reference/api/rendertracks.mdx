## Description
Renders the given list of tracks. 

## Signatures

<TypeTable>
    <TypeRow type="js" name="renderTracks(tracks: Track[]): void;" />
    <TypeRow type="net" name="void RenderTracks(IList<AlphaTab.Model.Track> tracks)" />
    <TypeRow type="android" name="fun renderTracks(tracks: alphaTab.collections.List<alphaTab.model.Track>): Unit" />
</TypeTable>

### Parameters 

<ParameterTable>
    <ParameterRow platform="js" name="tracks" type="alphaTab.model.Track[]">
        The tracks to render. They must all belong to the same score.
    </ParameterRow>
    <ParameterRow platform="net" name="tracks" type="IList<AlphaTab.Model.Track>">
        The tracks to render. They must all belong to the same score.
    </ParameterRow>
    <ParameterRow platform="android" name="tracks" type="alphaTab.collections.List<alphaTab.model.Track>">
        The tracks to render. They must all belong to the same score.
    </ParameterRow>
</ParameterTable>

### Returns 
Nothing
