### Description
Applies any changes that were done to the settings object. It also informs the <Signature style="inline" js={[["identifier","renderer","/docs/reference/types/alphatabapibase/renderer"]]} /> about any new values to consider.
By default alphaTab will not trigger any re-rendering or settings update just if the settings object itself was changed. This method must be called
to trigger an update of the settings in all components. Then a re-rendering can be initiated using the <Signature style="inline" js={[["identifier","render","/docs/reference/types/alphatabapibase/render"]]} /> method.
