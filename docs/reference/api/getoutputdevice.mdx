### Returns 
The custom configured output device which was set via <Signature style="inline" js={[["identifier","setOutputDevice","/docs/reference/types/alphatabapibase/#setoutputdevice"]]}>setOutputDevice</Signature>  or `null`
if the default outputDevice is used.
The output device might change dynamically if devices are connected/disconnected (e.g. bluetooth headset).
