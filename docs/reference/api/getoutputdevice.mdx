## Description
The currently configured output device if changed via [`setOutputDevice`](/docs/reference/plain/alphatabapibase.mdx#setoutputdevice). Assumes [`setOutputDevice`](/docs/reference/plain/alphatabapibase.mdx#setoutputdevice) has been used.
In the web version this functionality relies on experimental APIs and might not yet be available in all browsers. https://caniuse.com/mdn-api_audiocontext_sinkid

### Parameters 
None 

### Returns 
The custom configured output device which was set via [`setOutputDevice`](/docs/reference/plain/alphatabapibase.mdx#setoutputdevice) or `null`
if the default outputDevice is used.
The output device might change dynamically if devices are connected/disconnected (e.g. bluetooth headset).
