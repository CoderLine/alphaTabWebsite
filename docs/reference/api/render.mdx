## Description
Initiates a re-rendering of the current setup. If rendering is not yet possible, it will be deferred until the UI changes to be ready for rendering.

## Signatures

<TypeTable>
    <TypeRow type="js" name="function render()" />
    <TypeRow type="jquery" name="alphaTab('render')" />
    <TypeRow type="net" name="void Render()" />
</TypeTable>

### Parameters
None

### Returns
Nothing
