## Description
Initiates a load of the score using the given data. 

## Signatures

<TypeTable>
    <TypeRow type="js" name="load(scoreData: unknown, trackIndexes?: number[]): boolean;" />
    <TypeRow type="net" name="bool Load(object scoreData, IList<double>? trackIndexes)" />
    <TypeRow type="android" name="fun load(scoreData: Any, trackIndexes: alphaTab.collections.List<Double>?): Boolean" />
</TypeTable>

### Parameters 

<ParameterTable>
    <ParameterRow platform="js" name="scoreData" type="unknown">
        The data container supported by [`IUiFacade`](/docs/reference/plain/iuifacade.mdx).  The supported types is depending on the platform:
        
        * A `alphaTab.model.Score` instance (all platforms)
        * A `ArrayBuffer` or `Uint8Array` containing one of the supported file formats (all platforms, native byte array or input streams on other platforms)
        * A url from where to download the binary data of one of the supported file formats (browser only)
    </ParameterRow>
    <ParameterRow platform="net" name="scoreData" type="object">
        The data container supported by [`IUiFacade`](/docs/reference/plain/iuifacade.mdx).  The supported types is depending on the platform:
        
        * A `alphaTab.model.Score` instance (all platforms)
        * A `ArrayBuffer` or `Uint8Array` containing one of the supported file formats (all platforms, native byte array or input streams on other platforms)
        * A url from where to download the binary data of one of the supported file formats (browser only)
    </ParameterRow>
    <ParameterRow platform="android" name="scoreData" type="Any">
        The data container supported by [`IUiFacade`](/docs/reference/plain/iuifacade.mdx).  The supported types is depending on the platform:
        
        * A `alphaTab.model.Score` instance (all platforms)
        * A `ArrayBuffer` or `Uint8Array` containing one of the supported file formats (all platforms, native byte array or input streams on other platforms)
        * A url from where to download the binary data of one of the supported file formats (browser only)
    </ParameterRow>
    <ParameterRow platform="js" name="trackIndexes" type="number[]?">
        The indexes of the tracks from the song that should be rendered. If not provided, the first track of the
        song will be shown.
    </ParameterRow>
    <ParameterRow platform="net" name="trackIndexes" type="IList<double>?">
        The indexes of the tracks from the song that should be rendered. If not provided, the first track of the
        song will be shown.
    </ParameterRow>
    <ParameterRow platform="android" name="trackIndexes" type="alphaTab.collections.List<Double>?">
        The indexes of the tracks from the song that should be rendered. If not provided, the first track of the
        song will be shown.
    </ParameterRow>
</ParameterTable>

### Returns 
true if the data object is supported and a load was initiated, otherwise false
