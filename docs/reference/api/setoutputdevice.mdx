### Description
Changes the output device which should be used for playing the audio (player must be enabled). Use [`enumerateOutputDevices`](/docs/reference/types/alphatabapibase/enumerateoutputdevices) to load the list of available devices.

In the web version this functionality relies on experimental APIs and might not yet be available in all browsers. https://caniuse.com/mdn-api_audiocontext_sinkid
