### Description
Initiates a rendering of the given score.

<TabItem value="js">

</TabItem>
</Tabs>


<ParameterTable>
    <ParameterRow platform="all" name="score">
        The score containing the tracks to be rendered.
    </ParameterRow>
    <ParameterRow platform="all" name="trackIndexes">
        The indexes of the tracks from the song that should be rendered. If not provided, the first track of the
        song will be shown.
    </ParameterRow>
</ParameterTable>
