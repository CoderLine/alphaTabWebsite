### Description
Starts the playback of the current song.

<Signature style="block"
           js={[["identifier","play"],["token","("],["token",")"],["token",":"],["whitespace"," "],["identifier","boolean"]]}
           csharp={[["identifier","bool"],["whitespace"," "],["identifier","Play"],["token","("],["token",")"]]}
           kotlin={[["identifier","fun"],["whitespace"," "],["identifier","play"],["token","("],["token",")"],["token",":"],["whitespace"," "],["identifier","Boolean"]]}
/>

### Returns 
true if the playback was started, otherwise false. Reasons for not starting can be that the player is not ready or already playing.
