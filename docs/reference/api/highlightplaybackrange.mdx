Places the highlight markers at the specified start and end-beat range. Unlike actually setting <Signature style="inline" js={[["identifier","playbackRange","/docs/reference/types/alphatabapibase/#playbackrange"]]}>playbackRange</Signature>  this method only places the selection markers without actually
changing the playback range. This method can be used when building custom selection systems (e.g. having draggable handles).
