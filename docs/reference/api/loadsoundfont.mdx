## Description
This function initiates a load of the soundfont using the given data. The supported data types is depending on the platform. AlphaTab only supports SoundFont2 and SoundFont3 (<SinceBadge since="1.4.0" inline="true" />) encoded soundfonts for loading. To load a soundfont the player must be enabled in advance.

### Returns
`true` if the passed in object is a supported format and loading was initiated, otherwise `false`.
