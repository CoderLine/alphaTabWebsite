## Description
Triggers a load of the soundfont from the given data. AlphaTab only supports SoundFont2 and SoundFont3 <SinceBadge inline={true} since={"1.4.0"} /> encoded soundfonts for loading. To load a soundfont the player must be enabled in advance.

## Signatures

<TypeTable>
    <TypeRow type="js" name="loadSoundFont(data: unknown, append?: boolean): boolean;" />
    <TypeRow type="net" name="bool LoadSoundFont(object data, bool? append)" />
    <TypeRow type="android" name="fun loadSoundFont(data: Any, append: Boolean?): Boolean" />
</TypeTable>

### Parameters 

<ParameterTable>
    <ParameterRow platform="js" name="data" type="unknown">
        The data object to decode. The supported data types is depending on the platform.
        
        * A `ArrayBuffer` or `Uint8Array` (all platforms, native byte array or input streams on other platforms)
        * A url from where to download the binary data of one of the supported file formats (browser only)
    </ParameterRow>
    <ParameterRow platform="net" name="data" type="object">
        The data object to decode. The supported data types is depending on the platform.
        
        * A `ArrayBuffer` or `Uint8Array` (all platforms, native byte array or input streams on other platforms)
        * A url from where to download the binary data of one of the supported file formats (browser only)
    </ParameterRow>
    <ParameterRow platform="android" name="data" type="Any">
        The data object to decode. The supported data types is depending on the platform.
        
        * A `ArrayBuffer` or `Uint8Array` (all platforms, native byte array or input streams on other platforms)
        * A url from where to download the binary data of one of the supported file formats (browser only)
    </ParameterRow>
    <ParameterRow platform="js" name="append" type="boolean?">
        Whether to fully replace or append the data from the given soundfont.
    </ParameterRow>
    <ParameterRow platform="net" name="append" type="bool?">
        Whether to fully replace or append the data from the given soundfont.
    </ParameterRow>
    <ParameterRow platform="android" name="append" type="Boolean?">
        Whether to fully replace or append the data from the given soundfont.
    </ParameterRow>
</ParameterTable>

### Returns 
`true` if the passed in object is a supported format and loading was initiated, otherwise `false`.
