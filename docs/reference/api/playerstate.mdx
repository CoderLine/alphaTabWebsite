## Description
Gets the current player state, meaning whether it is paused or playing. 

## Types

<TypeTable>
    <TypeRow type="js" name="alphaTab.synth.PlayerState">
        `0` for paused<br />
        `1` for playing
    </TypeRow>
    <TypeRow type="net" name="AlphaTab.Synth.PlayerState" />
</TypeTable>
