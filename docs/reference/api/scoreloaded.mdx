### Description
This event is fired whenever a new song is loaded. This event is fired whenever a new song is loaded or changing due to [`renderScore`](/docs/reference/types/alphatabapibase/renderscore) or [`renderTracks`](/docs/reference/types/alphatabapibase/rendertracks) calls.
It is fired after the transposition midi pitches from the settings were applied, but before any midi is generated or rendering is started.
This allows any modification of the score before further processing.
