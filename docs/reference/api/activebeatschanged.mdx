### Description
This event is fired when the currently active beats across all tracks change. Unlike the <code style={{"color":"#393A34","backgroundColor":"#f6f8fa"}}><Link style={{"color":"#36acaa"}} to={"/docs/reference/types/alphatabapibase/playedbeatchanged"}>{"playedBeatChanged"}</Link></code> event this event contains the beats of all tracks and voices independent of them being rendered.
