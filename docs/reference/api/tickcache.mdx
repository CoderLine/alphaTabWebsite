## Description
The tick cache allowing lookup of midi ticks to beats. Gets the tick cache allowing lookup of midi ticks to beats. If the player is enabled, a midi file will be generated
for the loaded [`Score`](/docs/reference/plain/score.mdx) for later playback. During this generation this tick cache is filled with the
exact midi ticks when beats are played.

The [`MidiTickLookup.findBeat`](/docs/reference/plain/miditicklookup.mdx#findbeat) method allows a lookup of the beat related to a given input midi tick.
