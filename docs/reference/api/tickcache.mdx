### Description
The tick cache allowing lookup of midi ticks to beats. Gets the tick cache allowing lookup of midi ticks to beats. If the player is enabled, a midi file will be generated
for the loaded <code style={{"color":"#393A34","backgroundColor":"#f6f8fa"}}><Link style={{"color":"#36acaa"}} to={"/docs/reference/types/model/score/"}>{"Score"}</Link></code> for later playback. During this generation this tick cache is filled with the
exact midi ticks when beats are played.

The <code style={{"color":"#393A34","backgroundColor":"#f6f8fa"}}><Link style={{"color":"#36acaa"}} to={"/docs/reference/types/midi/miditicklookup/findbeat"}>{"findBeat"}</Link></code> method allows a lookup of the beat related to a given input midi tick.
