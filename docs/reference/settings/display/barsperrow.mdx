## Description
This setting sets the number of bars that should be put into one row during layouting. This setting is only respected
when using the [`LayoutMode.Page`](/docs/reference/plain/layoutmode.mdx#page) where bars are aligned in systems. [Demo](/docs/showcase/layouts#page-layout-5-bars-per-row).

<TypeTable>

</TypeTable>

## Default Value

`-1` (automatic)
