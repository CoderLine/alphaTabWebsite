Limit the displayed bars per system (row). (-1 for automatic mode) This setting sets the number of bars that should be put into one row during layouting. This setting is only respected
when using the <Signature style="inline" js={[["identifier","Page","/docs/reference/types/layoutmode#page"]]}>Page</Signature>  where bars are aligned in systems. [Demo](/docs/showcase/layouts#page-layout-5-bars-per-row).
