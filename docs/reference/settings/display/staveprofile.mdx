## Description
AlphaTab has various stave profiles that define which staves will be shown in for the rendered tracks.

`import * as alphaTab from '@coderline/alphaTab';`

* `alphaTab.StaveProfile.Default` - The profile is auto detected by the track configurations.
* `alphaTab.StaveProfile.ScoreTab` - Standard music notation and guitar tablature are rendered.
* `alphaTab.StaveProfile.Score` - Only standard music notation is rendered.
* `alphaTab.StaveProfile.Tab` - Only guitar tablature is rendered.
* `alphaTab.StaveProfile.TabMixed` - Only guitar tablature is rendered, but also rests and time signatures are not shown.
This profile is typically used in multi-track scenarios.
</TabItem>
