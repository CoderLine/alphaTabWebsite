## Description
AlphaTab has various stave profiles that define which staves will be shown in for the rendered tracks. Its recommended
to keep this on [`StaveProfile.Default`](/docs/reference/plain/staveprofile.mdx#default) and rather rely on the options available ob [`Staff`](/docs/reference/plain/staff.mdx) level

### alphaTab.StaveProfile
<TypeTable>
  <TypeRow type="js" name="alphaTab.StaveProfile">
    `Default` - The profile is auto detected by the track configurations.<br />
    `ScoreTab` - Standard music notation and guitar tablature are rendered.<br />
    `Score` - Only standard music notation is rendered.<br />
    `Tab` - Only guitar tablature is rendered.<br />
    `TabMixed` - Only guitar tablature is rendered, but also rests and time signatures are not shown.
  </TypeRow>
  <TypeRow type="json-js-html" name="string">
    `default`<br />
    `scoretab`<br />
    `score`<br />
    `tab`<br />
    `tabmixed`
  </TypeRow>
  <TypeRow type="json-js-html" name="int">
    `0` - Default <br />
    `1` - ScoreTab <br />
    `2` - Score <br />
    `3` - Tab <br />
    `4` - TabMixed 
  </TypeRow>
