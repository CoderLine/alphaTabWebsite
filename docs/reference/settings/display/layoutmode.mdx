## Description
AlphaTab has various layout engines that arrange the rendered bars differently. This setting controls which layout mode is used.

### alphaTab.LayoutMode
<TypeTable>
  <TypeRow type="js" name="alphaTab.LayoutMode">
    `Page` - The bars are aligned in an [vertically endless page-style fashion](/docs/showcase/layouts#page-layout)<br />
    `Horizontal` - Bars are aligned horizontally in [one horizontally endless system (row)](/docs/showcase/layouts#horizontal-layout)
  </TypeRow>
  <TypeRow type="json-js-html" name="string">
    `page`<br />
    `horizontal`
  </TypeRow>
