## Description

AlphaTab has various layout engines that arrange the rendered bars differently. This setting controls which layout mode is used. 

<PropertyDescription metadata={metadata} />

## Types

<TypeTable>
    <TypeRow type="js" name="alphaTab.LayoutMode">
        `Page` - The bars are aligned in a [page-style fashion](/docs/showcase/layouts#page-layout")<br />
        `Horizontal`  - The bars are aligned in a [left-to-right](/docs/showcase/layouts#horizontal-layout)
    </TypeRow>
    <TypeRow type="json-js-html" name="string">
        `page`<br />
        `horizontal`
    </TypeRow>
    <TypeRow type="json-js-html" name="int">
        `0` - Page<br />
        `1` - Horizontal
    </TypeRow>
    <TypeRow type="net" name="AlphaTab.LayoutMode">
        `Page`<br />
        `Horizontal`
    </TypeRow>
</TypeTable>

## Default Value

`page`
