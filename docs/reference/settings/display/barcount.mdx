### Description
The total number of bars that should be rendered from the song. This setting sets the number of bars that should be rendered from the overall song. This setting can be used to
achieve a paging system or to only show partial bars of the same file. By this a tutorial alike display can be achieved
that explains various parts of the song. [Demo](/docs/showcase/layouts)


### Default Value

`-1` (All bars)
