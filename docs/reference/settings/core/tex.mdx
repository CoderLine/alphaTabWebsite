### Description
Whether the contents of the DOM element should be loaded as alphaTex. This setting allows you to fill alphaTex code into the DOM element and make alphaTab automatically
load it when initializing. Note that this setting is only interpreted once on initialization.

<Signature style="block"
           js={[["identifier","tex"],["token",":"],["whitespace"," "],["identifier","boolean"],["whitespace"," "],["token","="],["whitespace"," "],["identifier","false"],["token",";"]]}
/>


## Example - JavaScript

```html
<div id="alphaTab">\title "Simple alphaTex init" . 3.3*4</div>
<script>
const api = new alphaTab.AlphaTabApi(document.getElementById('alphaTab'), { core: { tex: true }});
</script>
```
