## Description
AlphaTab internally does quite a bit of logging for debugging and informational purposes. The log level of alphaTab can be controlled via this setting.

<PropertyDescription metadata={metadata} />

## Types

<TypeTable>
    <TypeRow type="js" name="alphaTab.LogLevel">
        `None` - No logging<br />
        `Debug` - Debug level (internal details are displayed)<br />
        `Info` - Info level (only important details are shown)<br />
        `Warning`<br />
        `Error`
    </TypeRow>
    <TypeRow type="json-js-html" name="string">
        `none`<br />
        `debug`<br />
        `info`<br />
        `warning`<br />
        `error`
    </TypeRow>
    <TypeRow type="json-js-html" name="int">
        `0` - None<br />
        `1` - Debug<br />
        `2` - Info<br />
        `3` - Warning<br />
        `4` - Error
    </TypeRow>
    <TypeRow type="net" name="AlphaTab.LogLevel">
        `None`<br />
        `Debug`<br />
        `Info`<br />
        `Warning`<br />
        `Error`
    </TypeRow>
</TypeTable>

## Default Value

`Info`
