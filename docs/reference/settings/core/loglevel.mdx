### Description
The log level to use within alphaTab AlphaTab internally does quite a bit of logging for debugging and informational purposes. The log level of alphaTab can be controlled via this setting.

<Tabs defaultValue="js" values={[{label: "JavaScript", value: "js"},{label: "C#", value: "cs"},{label:"Kotlin", value: "kt"}]}>
<TabItem value="js">
<CodeBlock language="ts">{`logLevel : LogLevel = LogLevel.Info;`}</CodeBlock>
</TabItem>
<TabItem value="cs">
<CodeBlock language="csharp">{`AlphaTab.LogLevel LogLevel { get; set; } = LogLevel.Info`}</CodeBlock>
</TabItem>
<TabItem value="kt">
<CodeBlock language="kotlin">{`var logLevel: alphaTab.LogLevel = LogLevel.Info`}</CodeBlock>
</TabItem>
</Tabs>
