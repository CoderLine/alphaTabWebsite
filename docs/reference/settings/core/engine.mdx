## Description
AlphaTab can use various render engines to draw the music notation. The available render engines is specific to the platform. Please refer to the table below to find out which engines are available on which platform.    

<PropertyDescription metadata={metadata} showJson={true} />

## Types

<TypeTable>
    <TypeRow type="js" name="string">
        `svg` - Outputs SVG strings. <br />
        `html5` - Uses HTML5 canvas elements to render the music notation. 
    </TypeRow>
    <TypeRow type="net" name="string">
        `skia` - Available on all .net platforms. Uses [Skia](https://skia.org/) for rendering<br />
        `gdi` - Only available desktop .net. Uses [GDI+](https://docs.microsoft.com/en-us/dotnet/framework/winforms/advanced/graphics-and-drawing-in-windows-forms) for rendering <br /> 
        `svg` - Available on all .net platforms. Outputs SVG strings. 
    </TypeRow>
</TypeTable>

## Default Value

`default` which is mapped per platform: 

<TypeTable>
    <TypeRow type="js-html" name="svg" />
    <TypeRow type="net" name="skia" />
</TypeTable>
