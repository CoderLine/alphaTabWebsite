## Description
This setting enables the display of rhythm notation on tab staffs. [Demo](/docs/showcase/guitar-tabs)
<SinceBadge inline={true} since={"1.4.0"} /> its automatically detected whether rhythm notation should be shown on tabs (based on the visibility of other staves).

### alphaTab.TabRhythmMode
<TypeTable>
  <TypeRow type="js" name="alphaTab.TabRhythmMode">
    `Hidden` - Rhythm notation is hidden.<br />
    `ShowWithBeams` - Rhythm notation is shown with individual beams per beat.<br />
    `ShowWithBars` - Rhythm notation is shown and behaves like normal score notation with connected bars.<br />
    `Automatic` - Automatic detection whether the tabs should show rhythm based on hidden standard notation.
  </TypeRow>
  <TypeRow type="json-js-html" name="string">
    `hidden`<br />
    `showwithbeams`<br />
    `showwithbars`<br />
    `automatic`
  </TypeRow>
  <TypeRow type="json-js-html" name="int">
    `0` - Hidden <br />
    `1` - ShowWithBeams <br />
    `2` - ShowWithBars <br />
    `3` - Automatic 
  </TypeRow>
  <TypeRow type="net" name="AlphaTab.TabRhythmMode">
    `Hidden`<br />
    `ShowWithBeams`<br />
    `ShowWithBars`<br />
    `Automatic`
  </TypeRow>
  <TypeRow type="android" name="alphaTab.TabRhythmMode">

    `Automatic`
  </TypeRow>
</TypeTable>

## Default Value

`TabRhythmMode.Automatic`
