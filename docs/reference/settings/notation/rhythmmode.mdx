## Description

This setting enables the display of rhythm notation on tab staffs. [Demo](/docs/showcase/guitar-tabs)

<PropertyDescription showJson={true} />

## Types

<TypeTable>
    <TypeRow name="alphaTab.TabRhythmMode" type="js">
        `Hidden` -  No rhythm notation is shown<br />
        `ShowWithBeams` - Rhythm notation is rendered with all beats having individual beams<br />
        `ShowWithBars` - Rhythm notation is rendered with normal bars like on standard notation
    </TypeRow>
    <TypeRow name="string" type="json-js-html">
        `hidden`<br />
        `showwithbeams`<br />
        `showwithbars`
    </TypeRow>
    <TypeRow name="int" type="json-js-html">
        `0` - Hidden<br />
        `1` - ShowWithBeams<br />
        `2` - ShowWithBars
    </TypeRow>
    <TypeRow name="AlphaTab.TabRhythmMode" type="net">
        `Hidden`<br />
        `ShowWithBeams`<br />
        `ShowWithBars`
    </TypeRow>
    <TypeRow name="alphaTab.TabRhythmMode" type="android">
        `Hidden`<br />
        `ShowWithBeams`<br />
        `ShowWithBars`
    </TypeRow>
</TypeTable>

## Default Value

`hidden`
