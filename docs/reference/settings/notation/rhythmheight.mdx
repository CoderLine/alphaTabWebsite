### Description
Controls how high the ryhthm notation is rendered below the tab staff This setting can be used in combination with the [`rhythmMode`](/docs/reference/settings/notation/rhythmmode) setting to control how high the rhythm notation should be rendered below the tab staff.


### Default Value

`15`
