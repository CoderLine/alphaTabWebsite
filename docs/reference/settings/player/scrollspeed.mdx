### Description
How fast the scrolling to the new position should happen. If possible from the platform, alphaTab will try to do a smooth scrolling to the played bar.
This setting defines the speed of scrolling in milliseconds.
Note that <code style={{"color":"#393A34","backgroundColor":"#f6f8fa"}}><Link style={{"color":"#36acaa"}} to={"/docs/reference/settings/player/nativebrowsersmoothscroll"}>{"nativeBrowserSmoothScroll"}</Link></code> must be set to `false` for this to have an effect.
