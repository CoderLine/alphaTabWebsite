How fast the scrolling to the new position should happen. If possible from the platform, alphaTab will try to do a smooth scrolling to the played bar.
This setting defines the speed of scrolling in milliseconds.
Note that <Signature style="inline" js={[["identifier","nativeBrowserSmoothScroll","/docs/reference/settings/player/nativebrowsersmoothscroll"]]}>nativeBrowserSmoothScroll</Signature>  must be set to `false` for this to have an effect.
