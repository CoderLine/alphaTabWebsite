## Description
This setting controls how alphaTab behaves for scrolling.

* `alphaTab.ScrollMode.Off` - Do not scroll automatically
* `alphaTab.ScrollMode.Continuous` - Scrolling happens as soon the offsets of the cursors change.
* `alphaTab.ScrollMode.OffScreen` - Scrolling happens as soon the cursors exceed the displayed range.
