### [alphaTab.ScrollMode](/docs/reference/plain/scrollmode.mdx)
<TypeTable>
  <TypeRow type="js" name="alphaTab.ScrollMode">
    `Off` - Do not scroll automatically<br />
    `Continuous` - Scrolling happens as soon the offsets of the cursors change.<br />
    `OffScreen` - Scrolling happens as soon the cursors exceed the displayed range.
  </TypeRow>
  <TypeRow type="json-js-html" name="string">
    `off`<br />
    `continuous`<br />
    `offscreen`
  </TypeRow>
  <TypeRow type="json-js-html" name="int">
    `0` - Off <br />
    `1` - Continuous <br />
    `2` - OffScreen 
  </TypeRow>
  <TypeRow type="net" name="AlphaTab.ScrollMode">
    `Off`<br />
    `Continuous`<br />
    `OffScreen`
  </TypeRow>
  <TypeRow type="android" name="alphaTab.ScrollMode">
    `Off`<br />
    `Continuous`<br />
    `OffScreen`
  </TypeRow>
</TypeTable>

## Default Value

`ScrollMode.Continuous`
