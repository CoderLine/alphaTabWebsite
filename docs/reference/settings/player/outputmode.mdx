### alphaTab.PlayerOutputMode
<TypeTable>
  <TypeRow type="js" name="alphaTab.PlayerOutputMode">
    `WebAudioAudioWorklets` - If audio worklets are available in the browser, they will be used for playing the audio.<br />
    `WebAudioScriptProcessor` - Uses the legacy ScriptProcessor output which might perform worse.
  </TypeRow>
  <TypeRow type="json-js-html" name="string">
    `webaudioaudioworklets`<br />
    `webaudioscriptprocessor`
  </TypeRow>
  <TypeRow type="json-js-html" name="int">
    `0` - WebAudioAudioWorklets <br />
    `1` - WebAudioScriptProcessor 
  </TypeRow>
  <TypeRow type="net" name="AlphaTab.PlayerOutputMode">
    `WebAudioAudioWorklets`<br />
    `WebAudioScriptProcessor`
  </TypeRow>
  <TypeRow type="android" name="alphaTab.PlayerOutputMode">
    `WebAudioAudioWorklets`<br />
    `WebAudioScriptProcessor`
  </TypeRow>
</TypeTable>

## Default Value

`PlayerOutputMode.WebAudioAudioWorklets`
