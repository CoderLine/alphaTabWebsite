## `\title`

```plain title="Syntax"
\title (value template textAlign)
```

**Description:** Set the title of the song.

**Parameters:**

| Name | Description | Type | Required |
|------|-------------|------|----------|
|  `value` | The value to set | `String` | yes  |
|  `template` | The template used to render the text | `String` | no  |
|  `textAlign` | The alignment of the text on the music sheet | `left\|center\|right` | no  |

**Parameter Values:**

Following parameters have value limitations

| Name | Values |
|------|--------|
| `textAlign` | <ParameterValuesTable values={[["left","Left-align the text"],["center","Center-align the text"],["right","Right-align the text"]]} /> |

**Example:**

<AlphaTexSample tracks="all" player={true} >{`
\\title ("Song Title" "Title: %TITLE%" left)
C4

`}</AlphaTexSample>

## `\systemsLayout`

```plain title="Syntax"
\systemsLayout (numberOfBars)
```

**Description:** Defines the number of bars to display per system when rendering multiple tracks.

The `systemsLayout` and `defaultSystemsLayout` allow configuring the system layout. The system layout, defines how many bars should be displayed per system (line) if enabled via [`systemsLayoutMode`](/docs/reference/settings/display/systemslayoutmode).


**Parameters:**

| Name | Description | Type | Required |
|------|-------------|------|----------|
|  `numberOfBars` | Defines for every system (line) the number of bars it should contain | `Number[]` | no  |

**Example:**

<AlphaTexSample tracks="all" player={true} settings={{"display":{"systemsLayoutMode":"UseModelLayout"}}}>{`
\\systemsLayout (2 3 2)
\\track 
    :1 c4 | c4 | c4 | c4 | c4 | c4 | c4
\\track 
    :1 c4 | c4 | c4 | c4 | c4 | c4 | c4

`}</AlphaTexSample>
